# KeyBox3D

**Web 端 LiDAR 3D 框标注 & 关键点标注工具**

KeyBox3D 是一个轻量级的 Web 端 LiDAR 3D 点云标注工具。与大多数仅支持框标注的工具不同，KeyBox3D 原生支持 **3D 框标注与关键点标注** — 是少数具备此能力的开源工具之一。

**为什么选择 KeyBox3D？**
- 🎯 **框 + 关键点**：在一个工作流中标注 3D 框和关联关键点 — 关键点自动绑定到父级框
- 🖥️ **四视图联动**：主 3D 视图 + 俯视/侧视/主视正交辅助视图，精确标注
- 🤖 **智能辅助**：自动适配 (K) 收缩框到点云边界；地面贴合 (G) 自动对齐地面
- 📦 **KITTI + 关键点导出**：扩展标准 KITTI 格式，附带关键点世界坐标
- ⚡ **零配置**：纯 Web 应用 — `pip install` 后打开浏览器即用，无需 GPU
- 🔗 **序列支持**：帧传播、残影显示、标注进度跟踪

## ✨ 核心特性

*   **专业可视化**:
    *   **四视图**: 主 3D 视图 + 俯视/侧视/正视 三个正交辅助视图。
    *   **高级渲染**: 支持 高度/强度 着色模式切换，地面过滤，圆形点云渲染。
    *   **情境感知**: 鼠标悬停高亮（带粗边框），框内点云实时高亮。
*   **智能辅助**:
    *   **自动适配 (K)**: 自动收缩框体以紧贴内部点云（带可配置边距）。
    *   **地面贴合 (G)**: 自动将框体底部吸附到地面平面。
    *   **序列传播**: 一键将当前帧标注复制到下一帧。
    *   **残影显示**: 显示上一帧的标注残影，辅助轨迹跟踪。
*   **高效工作流**:
    *   **严格模式**: 独立的 Box / Point 模式切换，防止类别混用。
    *   **类别管理**: 自定义类别及其默认尺寸 (L/W/H)。
    *   **自动关联**: 点 (Point) 会自动关联到最近操作的框 (Box)。
    *   **轴向锁定**: 锁定 Z 轴旋转，防止框体倾斜。
    *   **批量转换**: 支持 JSON 到 KITTI 格式的批量转换。
    *   **数据完整性**: 保存时自动同步多格式；列表显示标注进度（绿/灰圆点）。

## 🛠️ 安装指南

### 前置要求
*   **Python 3.8+**
*   **现代浏览器** (Chrome/Edge/Firefox)

### 步骤

1.  **克隆或下载** 本仓库。
2.  **安装 Python 依赖**:
    ```bash
    pip install -r requirements.txt
    ```
    *核心依赖: `fastapi`, `uvicorn`, `numpy`, `open3d` (可选，用于支持 PCD).*

3.  **启动服务器**:
    ```bash
    python backend.py
    ```
    服务器将在 `http://127.0.0.1:8000` 启动。

## 📖 使用指南

### 1. 数据加载
*   在浏览器打开 `http://127.0.0.1:8000`。
*   在左上角 **“数据目录”** 输入框中，填入数据集的绝对路径 (例如 `/home/user/data/kitti/2011_09_26/`)。
    *   *注：工具会自动搜索 `lidar`, `velodyne`, `pointcloud` 等子目录。*
*   点击 **“加载目录”**。
*   **进度圆点**:
    *   ⚪ 灰色: 未标注。
    *   🟢 绿色: 已标注 (存在 JSON)。

### 2. 基本交互
*   **视角操作**:
    *   **左键拖拽**: 旋转。
    *   **中键 / 右键拖拽**: 平移 (推荐使用中键以避免浏览器手势冲突)。
    *   **滚轮滚动**: 缩放 (支持极近距离观察)。
*   **工具**:
    *   **W (Box模式)**: 切换到 Box 类别列表并添加框。
    *   **S (Point模式)**: 切换到 Point 类别列表并添加点。
    *   **1 / 2 / 3**: 切换 **移动 / 旋转 / 缩放** 模式。
    *   **Esc**: 取消选中 / 自由浏览模式。
    *   **Del**: 删除选中对象。
*   **可视化惯例**:
    *   **绿色框**: 标注对象。
    *   **青色箭头**: 指示车头方向 (Local +Y 轴)。
    *   **绿色轴 (Gizmo)**: 代表正前方。

### 3. 智能工具与快捷键
*   **G (地面贴合)**: 选中框 -> 按 `G`。框体会自动贴合到下方的地面。
*   **K (自动适配)**: 选中框 -> 按 `K`。框体会根据内部点云自动收缩尺寸（+边距）。
*   **传播 (>>>)**: 点击“传播到下一帧”按钮，将当前所有框复制到下一帧。
*   **M**: 显示/隐藏 类别统计直方图。
*   **? 或 /**: 显示/隐藏 快捷键备忘录。

### 4. 配置与导出
*   **类别管理**:
    *   **Box / Point 独立管理**: 可以在两种模式下分别创建和管理类别。
    *   **导入/导出**: 将配置保存为 JSON 文件以便复用。
*   **导出格式**:
    *   **KITTI 包含关键点**: 
        *   ✅ **开启 (默认)**: 自动检测。如果框内含关键点，导出的 KITTI 文件将追加关键点坐标 (x y z ...)，并对齐补零。
        *   ⬜ **关闭**: 强制截断。仅导出标准 KITTI 15列数据，忽略所有关键点。
    *   **自动同步**: 保存时会自动更新 `label_json/` 和 `label_kitti/` 下的文件。
    *   **批量转换**: 切换格式时会触发全量数据的批量转换。

## 🔧 辅助工具脚本

项目根目录下提供了两个实用的 Python 脚本，用于数据迁移和格式增强导出。

### 1. `convert_x_to_y.py` (数据迁移)
**功能**: 将旧版本（X轴为车头）的 JSON 标注文件批量转换为新版本（Y轴为车头）。
**操作**: 自动交换长宽 (`scale.x` <-> `scale.y`) 并旋转朝向 (-90°)。
**用法**:
```bash
python convert_x_to_y.py
```
*注：脚本会自动检测 `dataset_bin_output/label_json` 目录并创建备份。*

### 2. `json_to_kitti_with_keypoints.py` (增强导出)
**功能**: 将 JSON 标注转换为 **"KITTI + 关键点"** 的扩展格式。
**输出格式**: 
`Type ... H W L X Y Z Ry [K1_x K1_y K1_z] [K2_x K2_y K2_z] ...`
*   前 15 列：标准 KITTI 3D 框格式。
*   后续列：该框关联的关键点世界坐标。
**用法**:
```bash
python json_to_kitti_with_keypoints.py
```
*输出目录: `dataset_bin_output/label_kitti_keypoints`*
